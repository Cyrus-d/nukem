Patterns

Domain Names

The domain name is matched first and the pattern is applied to the rest of the URL. Wildcards can also be used in the domain to match multiple domains or subdomains, eg. will match groups.google.com and maps.google.co.uk.

Help with patterns

The default pattern matches URLs that are pages within the website but not the main page of the website. If you want to match particular pages you can modify the pattern. For example, the pattern will only match pages with do not end with a #.
